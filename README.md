# 使用SSM的家居购项目

#### 技术栈：

##### 说明：前后端分离开发，前端框架Vue + 后端框架SSM

1、前端框架 Vue

2、后端框架 SSM （SpringMVC + Spring +MyBatis）

3、数据库 MySQL

4、项目依赖管理 Maven

5、 分页 pagehelper

6、 逆向工程 MyBatis Generator

### 项目说明

#### 简单的家居购物后台管理，具有增删改查、分页以及数据验证的功能

前端页面较为简单，仅具有展示数据美化的功能，没有较多美化的CSS

### 运行环境

#### 前端 Node Js 版本：14.17.3

#### 后端 TomCat 版本：8.5.91



未给出相应数据库代码，需要自己建立一个**furns_ssm**数据库，更改相关数据库配置文件
